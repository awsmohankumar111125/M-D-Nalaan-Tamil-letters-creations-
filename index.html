📚 How Tamil Letter Joining Works

  • Consonants (மெய் எழுத்து) need a special mark (pulli · புள்ளி) to show they don't have the default 'a' sound
  • Vowels (உயிர் எழுத்து) attach as marks to change the consonant's sound
  • The result is called uyirmei (உயிர்மெய்) — a compound letter
  • Example: க் (k) + ஓ (oo) = கோ (kō)
